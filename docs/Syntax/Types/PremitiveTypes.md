## プリミティブ型

### int型
符号付き32ビット数値

- C言語の型: int32_t (C99)
  - ヘッダー: stdint.h

### uint型
符号なし32ビット数値

- C言語の型: uint32_t (C99)
  - ヘッダー: stdint.h

### byte型
符号付き8ビット数値

- C言語の型: int8_t (C99)
  - ヘッダー: stdint.h

### ubyte型
符号なし8ビット数値

- C言語の型: uint8_t (C99)
  - ヘッダー: stdint.h

### bool型
`true`と`false`の2値

- C言語の型: _Bool (C99)
  - ヘッダー: stdbool.h

### void型
値がないことを示す型

- C言語の型: void (C89)
