## プリミティブ型

### int8型
符号あり8ビット数値

- C言語の型: `int8_t` (C99)
  - ヘッダー: stdint.h

### uint8型
符号なし8ビット数値

- C言語の型: `uint8_t` (C99)
  - ヘッダー: stdint.h

### int16型
符号付き16ビット数値

- C言語の型: `int16_t` (C99)
  - ヘッダー: stdint.h

### uint16型
符号なし16ビット数値

- C言語の型: `uint16_t` (C99)
  - ヘッダー: stdint.h

### int32型
符号付き32ビット数値

- C言語の型: `int32_t` (C99)
  - ヘッダー: stdint.h

### uint32型
符号なし32ビット数値

- C言語の型: `uint32_t` (C99)
  - ヘッダー: stdint.h

### int64型
符号付き64ビット数値

- C言語の型: `int64_t` (C99)
  - ヘッダー: stdint.h

### uint64型
符号なし32ビット数値

- C言語の型: `uint64_t` (C99)
  - ヘッダー: stdint.h

### bool型
`true`と`false`の2値

- C言語の型: `_Bool` (C99)
  - ヘッダー: stdbool.h

### void型
値がないことを示す型

- C言語の型: `void` (C89)
