# リテラル

## 数値リテラル
- 数値リテラルは数値を返します。
- 型推論は要検討

```
var x: int = 123;
var y: byte = 32;
8;
```
生成コード:
```c
#include <stdint.h>

// ...

int32_t x = 123;
uint8_t y = 32;
8;
```


## 文字リテラル
- 文字リテラルはchar型の値を返します。
- 生成されるコードでは、文字リテラルは_char構造体の値を生成します。

```
var x: char = 'a';
```
生成コード:
```c
// 未定
```


## 文字列リテラル
- 文字列リテラルはstring型の値を返します。
- 生成されるコードでは、_char構造体の配列を生成します。

```
var x: string = "abc";
```
生成コード:
```c
// 未定
```


## null
空のポインタを示す値。

- C言語の型: NULL (C89)

```
var x = null;
```
生成コード：
```c
#include <stddef.h>

// ...

int *x = NULL;
```
